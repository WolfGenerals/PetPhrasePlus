# Pet Phrase Plus

__在玩家发出的信息末尾加上一段文字，或者替换其中的某些文字！__  

---
## 前置
Fabric Language Kotlin
Cloth Config API
## 配置文件  
配置文件位于`config/petphraseplus.json`  
可以用指令`/petphraseplus`打开配置界面  
也可以通过Modmenu模组打开配置界面

### replace  
用来替换的关键字  
示例：`我/咱`会将聊天信息中的所有“我”替换为“咱”  

### start
前缀

### endInner  
后缀，放在标点符号内

### endOuter  
后缀，放在标点符号外
